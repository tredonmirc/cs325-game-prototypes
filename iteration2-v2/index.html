  • Easiest or hardest thing
    • This game has been a challenge for several reasons. The first rose from my attempt to create a tilemap based board game that had better movement than the first (Dog Walker) game. This let me down the path of implementing movement with static x,y updates (instead of "velocity" type movements). ALthough this provides exact movement it prevents the use of collision because the sprits are "moving" into others but effectively teleport. This was solved by laboriously mapping each tile location to an array and marking which are valid tiles for movement. Another issue I encountered was swapping sprite locations based on the tile locations. Though the location swap was pretty straightforward, gaurding the process to only occur once in a move took some ingenuity.
    • Another challenge IS (some race cases seems to plaque the gameplay) ensuring accurate teleportation - occasions occur where the event doesn't trigger :(
    • Applying sounds to movement is REALLY easy in Phaser

  • Gameplay:

    The concept of this game is simple. You are a student, and Friday Night has finally come. You have options tonight - you can either Party (the pink path and score tiles), Work (the green path and score tiles) or study (the blue path and score tiles). You gain points but landing on score or Risk tiles. The score tiles will always give points, and the more you get in a specific type, the more points it awards. Think of it like added gains by "focusing" on a particular task. You will get, 1, then 2, then 4 and 8 points for each type of score tile you land on. RISK tiles give you a chance at obtaining points from paths other than the one you're on BUT they run a risk. If you roll a 1-2 on a 1d6, you get BONUS points, but 3-6 will subtract points!!! SWAP tiles let you be crafty and switch positions with the other player. Once a player uses a SWAP tile they cannot use it again. The first person to finish will terminate the game and the current points will determine who wins!

  • Strategies:

    Players will have to determine which path to persue, forcing some strategy with SWAPs and randomness with high risk, high rewards in the RISK tiles
